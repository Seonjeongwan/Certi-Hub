# 📋 Certi-Hub 서비스 개요

> IT 자격증 통합 관리 플랫폼 — 130개+ 자격증 정보, 시험 일정, 로드맵을 한 곳에서

---

## 🎯 서비스 소개

**Certi-Hub**는 IT 분야 자격증 정보를 통합적으로 관리하고, 시험 일정을 한눈에 확인할 수 있는 웹 플랫폼입니다. Cloud, AI, 데이터, 보안, 네트워크 등 다양한 분야의 **130개 이상의 IT 자격증**을 분류·비교하고, 시험 접수 기간부터 합격 발표일까지의 일정을 캘린더로 시각화합니다.

### 핵심 가치

- **통합 관리**: 흩어져 있는 각종 자격증 기관(Q-Net, KData, AWS, Google 등)의 시험 일정을 하나의 플랫폼에서 확인
- **자동 업데이트**: 6개 크롤러가 매일 새벽 자동으로 최신 시험 일정을 수집
- **로드맵 제공**: 레벨별(초급→중급→상급→고급) 자격증 경로를 시각적으로 제시
- **무중단 서비스**: 3단계 Fallback 전략으로 API 장애 시에도 캐시 데이터로 서비스 지속

## ✨ 주요 기능

### 1. 자격증 정보 조회 및 검색

- **130개+ IT 자격증** 데이터베이스
- **14개 대분류** 카테고리: Cloud, AI, 데이터, 서버/DB, 네트워크, 보안, 아키텍처, UX/UI, QA, Infra, Project Managing, Solution, 감사, 금융/기타
- **4단계 레벨 분류**: Basic(초급), Intermediate(중급), Advanced(상급), Master(고급)
- **실시간 자동완성 검색**: 한글/영문 자격증명, 분류명으로 즉시 검색
- **공식 접수 페이지 링크**: 각 자격증의 공식 시험 접수 URL 제공

### 2. 시험 일정 캘린더

- **FullCalendar 연동**: 월별/리스트 뷰 전환 가능
- **컬러 코딩**: 접수 기간(🔵파랑), 시험일(🔴빨강), 발표일(🟢초록) 3색 구분
- **카테고리 필터**: 원하는 분야만 선별하여 일정 확인
- **반응형 디자인**: 모바일에서는 자동으로 리스트 뷰 전환

### 3. 자격증 로드맵

- **테이블 뷰**: 대분류/소분류별로 레벨(초급→중급→상급→고급) 매핑 표
- **카드 뷰**: 자격증별 상세 카드 (태그, 레벨, 공식 URL)
- **레벨별 필터링**: 특정 레벨만 선별하여 확인
- **관련 자격증 추천**: 같은 분야의 인접 레벨 자격증 자동 추천

### 4. 자격증 상세 모달

- **기본 정보**: 국문/영문명, 분류, 레벨, 공식 URL
- **다가오는 시험 일정**: 해당 자격증의 가장 가까운 시험/접수 일정 표시
- **관련 자격증**: 같은 분야 내 다른 레벨 자격증 추천
- **공식 페이지 바로가기**: 원클릭으로 시험 접수 페이지 이동

### 5. 인기 자격증 대시보드

- **분야별 인기 자격증** 그리드: 카테고리별 대표 자격증 빠른 접근
- **통계 정보**: 총 자격증 수, 분야 수, 레벨 수, 시험 일정 수 실시간 표시

### 6. 자동 데이터 수집 (크롤러)

- **6개 전문 크롤러** 운영 (Q-Net, KData, Cloud, Finance, IT Domestic, International)
- **매일 새벽 3시** 자동 크롤링 (APScheduler)
- **3단계 Fallback 전략**: 공식 API → 웹 크롤링 → 캐시 데이터
- **크롤링 이력 관리**: 성공/실패 기록, 통계 대시보드

## 📂 프로젝트 구조 (Top-Level)

```

├── backend/              # FastAPI 백엔드 (Python)

├── .github/workflows/    # GitHub Actions CI/CD
└── .env.example          # 환경변수 예시
```
